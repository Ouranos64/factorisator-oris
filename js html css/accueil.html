À quoi sert donc le factorisator,oris ?

Le factorisator,oris est un programme qui permet de donner une forme factoriser à un polynôme. Pour l'instant, il ne peut que pour des polynômes de degré 1, 2, et 3. Mais les progrès ne sont pas exclus !

Comment marche donc le factorisator, oris ?

Pour utiliser le factorisator, oris, il faudra d'abord choisir le degré du plolynôme que vous souhaitez factoriser. \n Pour les polynômes de degré 2, vous devrez ensuite entrer leur coefficient. \n Pour les polynômes de degré 3, vous devrez entrer en plus des coefficient une racine connue du polynôme, c'est à dire une valeur pour laquelle le polynôem s'annule
